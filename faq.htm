Where can I get the latest Hero Lab FAQ information?

You should be able to find the information you need at the Hero Lab web site. The information there may be more up to date than this FAQ, so if your question isn't answered here, check there too.

I want to add my own custom game content. How do I do that?

Go to the "Tools" menu and select "Launch Editor". A detailed manual that covers adding your own content should be available by clicking the "Help" menu within the Editor.

How were these data files written and how can I extend them?

A detailed walk-through of how the Savage Worlds data files were developed is provided as part of the Hero Lab Authoring Kit. Complete documentation on the Authoring Kit will be found within the wiki dedicated to that purpose. Alternately, you can refer directly to the Savage World Walk-Through.

What changed in version 5.7 of the data files?

Enhancements & Changes

  • --- Core Updates ---
  • Stackable Armor now stacks correctly with standard armor rather than replacing it.
  • Cat, Small now has the description included.
  • The Fleet-Footed Edge has been moved to Background Edges.
  • Hide.Edge and Hide.Hindrance tags have been created, and filters for them are now active on the Edges tab.
  • A Derived Trait has been added for Running Die, and it can be turned on and off (it defaults to off) in the Configure Your Character window and can be adjusted through the Derived Trait Personal Adjustment. It will take a while for everything that adjusts Running Die to catch up with automatically adjusting it.
  • The list of affected objects is as follows (these all automatically adjust the value):
    • Core
      • Creature Abilities: Fleet-Footed (d8); Fleet-Footed (d10); Slow (Pace 4)
      • Edges: Fleet-Footed
      • Hindrances: Lame; Obese; One Leg
      • Racial Properties: Severly Reduced Pace
    • Fantasy Companion
      • Creature Abilities: Shuffling Gait; Fleet-Footed (d12)
      • Magic Items: Wolfskin Boots
    • Horror Companion
      • Creature Abilities: Scuttle; Lame; Obese; Shuffling Gait; Hop (Savage Jack); Extended Jump
    • Sci-Fi Companion
      • Creature Abilities: Fast; Leg Enhancements; Aquatic
      • Cyberware: Leg Enhancement
      • Edges: Heavy-G Worlder
      • Modifications: Power Armor - Pace
      • Racial Properties: Pace; Slow -2; Slow -4
      • Robotics: Pace
  • A Derived Trait has been added for Size, and it can be turned on and off (it defaults to off) in the Configure Your Character window and can be adjusted through the Derived Trait Personal Adjustment. It will take a while for everything that adjusts Size to catch up with not having to adjust Toughness separately.
  • The list of affected objects is as follows (these all adjust Toughness using Size if available):
    • Core
      • Creature Abilities: Size
    • Super Powers Companion
      • Powers: Growth
    • Sci-Fi Companion
      • Racial Properties: Size +1; Size +2; Size +3; Small -1; Small -2
      • Robotics: Size Increase
  • --- Science Fiction Companion updates ---
  • The Parry in the Serran Race template has been fixed.
  • --- Super Powers Companion updates ---
  • The updated 1st Edition villains are now included in the 2nd Edition stock villains files.
  • Attack, Ranged now shows adjusted damage and range for the Enhanced Damage and Range modifications.
  • Super Skills cost has been corrected.

What changed in version 5.5 of the data files?

Enhancements & Changes

  • Maximum Wound Penalty is now 3.
  • Custom Skills now omit the phrase "Custom [Attribute] Skill:" from printouts.
  • Custom Skills now correctly size long names up to 36 characters
  • Arcane Powers are no longer unique, to allow different versions of the Powers on the same character.
  • There is now a Permanent Adjustment for the number of Languages.
  • --- Fantasy Companion updates ---
  • Updated to match the current Fantasy Companion version.
  • Guts has been removed and replaced with Fear Checks.
  • Light and Obscure have been combined.
  • The Adaptable Edge has been fixed.
  • Adjustable magic items work correctly now.
  • --- Horror Companion updates ---
  • Relentless Edge has been corrected.
  • --- Sci-Fi Companion updates ---
  • Max Strain calculation has been corrected.
  • Power Armor values now count as Stackable Personal Armor.
  • --- Super Powers Companion updates ---
  • The first paragraph for the Malfunction power was re-formatted.
  • Stun text has been corrected.
  • Summary for Deflect modifier on Parry power has been corrected.

What changed in version 5.2 of the data files?

Enhancements & Changes

  • --- Core Updates ---
  • Corrected Switchable validation to properly account for the primary total
  • Added standard gear tags to the NPC gear from Super Powers Companion 1st Edition, and corrected the price on the telescopic baton
  • Corrected the activation on the Hard of Hearing Hindrance
  • Added user activation to the Phobia and Terminally Ill Hindrances
  • Journal tab was altered to allow decimal number in the Add Cash field
  • Equippable checkbox for equipment now displays correctly
  • Expansions to several editor pages to allow easier data entry
  • Fixed an issue with the multiplier display for stackQty in the code that builds the list of Modifications for display, both in the description block and the printed sheet. The (x#) display was being carried over into the next item.
  • Added the Standard Equipment tags to Particle Accelerators.
  • Added the Standard Equipment tags to some weapons that were hiding in the sf_creatures.dat file.
  • Corrected the spelling on a couple of magic item names (Boots of Speed and Salamander Gloves).
  • Cleaned up and re-ordered the Weapon Types, Vehicle Types and Gear Types.
  • Added a block to the Vehicle descriptions to display the Special Traits.
  • Fixed the weight display in the description to show fractional weights.
  • Corrected the Thrown modifier in Super Powers Companion to transfer properly to the Ranged Attack Power weapon object.
  • Fixed recent overflow additions to be appropriate when the hero is a creature.
  • Fixed the LoadOut on the AH-64 Apache helicopter.
  • Changed the way weapon parry bonuses are applied so the hero only receives the benefit of the one best bonus, and adding multiple weapons has no effect.
  • Fixed the Stock Robots file for the Sci-Fi Companion to remove extraneous items.
  • Changed the Gear List on the printed sheet from 3 columns to 2 to allow more text.
  • The Zombie Arcane Power Rank requirement has been corrected.
  • The summary on Long Stride (SPC) has been corrected.
  • Added some more text to Super Edge and Super Attribute (SPC) on the In-Play tab to make multiple instances easier to tell apart.
  • Added a new category on the printed sheet for gear that's Equippable in order to display the summary to describe what changes when it's equipped.
  • A new Stock character file for the Fantasy Companion has been created to remove the Guts Skill for SWD.
  • --- Programming Updates ---
  • Hindrance tags get pushed out to the hero earlier.
  • Changed the filters on form_sf_modifications.dat to stop using source tags to enable things and move to the Equipment.ModSlotUse tag or a value in Modification Slots Used. Using source tags caused problems in other Settings.
  • Created a tag (Equipment.ModSlotUse) to indicate if a weapon is allowed to be used in the Sci-Fi Companion-style modifications slots and doesn't have a value for Slots Used.
  • Added a checkbox to the Settings Editor to enable/disable Standard Modifications using the ModSlotUse tag above.
  • Changed the Derived Traits Current Strain, Max Strain and Mod Slots in the Sci-Fi Companion to read hero-level tags in order to be enabled rather than source tags.
  • Changed Cyber Resistant from the Sci-Fi Companion to check hero-level tags in addition to it's source tag.
  • Added a tag set (Equipment.StdMod) and exclusion tag (Hero.NoStdMods) to indicate whether to include the standard Modification set from the Sci-Fi Companion.
  • Added a tag set (Equipment.StdCyber) and exclusion tag (Hero.NoStdCyber) to indicate whether to include the standard Cyberware set from the Sci-Fi Companion.
  • Added a tag set (Equipment.StdRoMod) and exclusion tag (Hero.NoStdRoMod) to indicate whether to include the standard Robotic Modification set from the Sci-Fi Companion.
  • Added a hero field (acCshSymbl) and a tag (Hero.CashBackwd) to support variable cash symbols.
  • --- Interface Zero 2.0 Support Updates ---
  • Added the Athletics Skill and a Setting flag to replace the Climbing and Swimming Skills with the Athletics Skill.
  • Added Firewall rating as a field for vehicles for Interface Zero 2.0.
  • Added checkboxes to the Armor tab in the Editor to indicate covered body Locations.
  • Added the ability to enable User Activation on Cyberware for use with conditional modifiers. This is different than equipping/unequipping a unit like a Skill Chip as Strain remains unaffected, and is currently mutually exclusive with equipping/unequipping (one or the other).
  • Added 6 new Gear Types (Engrams, Drones, Drugs, Modules, Entertainment, and Delivery Subscriptions).
  • Added 7 new Weapons Types (Futuristic Pistols, Futuristic SMGs, Futuristic Rifles, Futuristic Shotguns, Futuristic Energy, Heavy Weapons, Vehicle Melee).
  • Added 3 new Armor Types (ExoArmor, Personal, Other).
  • Added 5 new Vehicle Types (Cars, Cycles, VTOLs, Buses & Trucks, Heavy Vehicles).
  • Added 5 new Weapon Mod types to cover the different types of melee weapon mods allowed in IZ2.0 (Retractable, Reach, Armor Piercing, Parry, Shock) for Interface Zero 2.0.
  • Added 10 new Hero fields to store various totals for allowable cyberware and software (engrams).

What changed in version 5.1 of the data files?

Enhancements & Changes

  • Overflow has been expanded on the printed sheets to include Derived Traits, Skills, Racial Abilities, Racial Properties, Hindrances, Edges and Creature Abilities
  • The Hard of Hearing Hindrance has been corrected to subtract from Notice
  • The Edge Killer Instinct has been added
  • The Professional and Expert Edges now apply their bonuses in all Settings
  • Equipment on the Gear list can now display a checkbox to choose Equipped status by having the Equipped.Equippable tag attached to it. Exoskeletons from the Sci-Fi Companion now use this to activate their Strength modifications.
  • The Hero Statblock Major Hindrances now show a (*) symbol to indicate Major Hindrances
  • The Scholar Edge now properly uses the stacking rules for professional edges
  • Allies would be printed incorrectly on the character sheet
  • [FC] The Barbaric Blood and Scamper Edges now work with the Adaptable Edge
  • [FC] Magic Weapon attack bonuses now apply exclusively to that weapon
  • [HC] The "Angel" Race in the Horror Companion has had the Arcane Background: Miracles removed
  • [SFC] The Standard Gear tag has been added to all stock equipment from the Sci-Fi Companion
  • [SFC] Crew Reduction for Starships has been changed to calculate additional modification slots correctly
  • [SFC] The Particle Accelerator vehicle weapons on page 25 of the Sci-Fi Companion are available now
  • [SFC] Personal Force Fields now stack with Armor - Programmer's Note: A new tag has been created, Equipment.StackArmor, that allows Defense values to stack on top of the normal Armor value
  • [SPC] Edge: Scholar (Worm) has the correct source set now
  • [SPC2] The Switchable Power Modifier from Super Powers Companion now works properly when evaluating cost of Alternate Slots versus Base Slots
  • [SPC2] The Super Edge: Trademark Weapon and Super Edge: Improved Trademark Weapon powers were added as separate entries to handle the extra selection of the weapon
  • [SPC2] Invention is now a selection under Power Modifiers, in the same fashion as Sorcery, to zero out the cost
  • [SPC2] The Matter Control power now allows for the Power Type to be specified
  • [SPC2] The Armor Power has had the Power Level descriptions corrected, along with some other Powers that had their descriptions edited a bit
  • [SPC2] The Power Modifier Limitations is now not unique, and can be taken multiple times
  • [SPC2] The Ranged Touch and Projectile power modifiers have been added to the Healing Power
  • [SPC2] The Champion, Mentalist, and Mr. Fix It Edges now do not require the Arcane Skills
  • [Data file authors] The EdgeStop architecture has been removed. To enable Edges to be turned on and off previously, this existed at the Edge level to allow the effects of Edges to be turned on and off. This has been replaced by requiring some additional code on eval scripts attached to Edges. Simply add "doneif (field[abilActive].value = 0)" as the first line in any Edge eval script.

What changed in version 2.6 of the data files?

Core

  • The printed character sheet now displays Languages consistent with the changes in Linguist and Multiple Languages in the previous update.
  • Some Savage Worlds Deluxe Vehicles were missing the Standard Gear tag, preventing them from being removed by the No Standard Gear Setting Adjustment. This has been fixed.
  • Corrected the Adept Edge pre-requisites and description to match errata from Pinnacle regarding it's pre-requisites and effects.
  • Corrected issues with Linguist and Multiple Languages, also correcting the number of languages allowed to include a slot for the Hero's native language.
  • Added Setting Adjustments for Born a Hero and No Power Points
  • Fixed Laser Sword Attack and Damage Bonuses
  • Fixed the Source on Sickle so it'll only show up when it's supposed to.
  • Added the Standard Gear tag to Mace, Scythe and Whip so they are excluded along with other Standard Gear.
  • Add list of the Hero's current weapons to a drop-down menu for Trademark Weapon Edge, and it's adds the bonus to the weapon.
  • Added User-Selectable modifiers for the following Edges and Hindrances:
    1. Champion, Double Shot, Exorcist, Frenzy, Giant Killer, Investigator, Liquid Courage, Marksman, One of the Chosen, and Sunder;
    2. Allergy, Doubting Thomas, Hard of Hearing (Major), Phobia, and Terminally Ill
    These will now appear on the In-Play tab under Activated Abilities. When selected, their respective bonuses/penalties will be applied to the hero.
  • The core version of the quiver was not set as a container.

Fantasy Companion

  • Corrected the Fantasy Companion Edges Scamper, Sunder and Improved Sunder to remove the erroneous starting cash modifier.
  • Corrected the Fantasy Companion version of Saurian to differentiate it from the Savage Worlds Deluxe version.
  • Added a Fantasy Companion version of Rakashan with the slight differences from the SWD version.
  • Corrected the issues with Fantasy Races being displayed at the wrong times.
  • Fixed the issue with magic shields where it was showing an error message about multiple shields being equipped whenever a single magic shield was equipped.
  • The fantasy version of the quiver was not set as a container.

Super Powers Companion

  • Added full support for the Switchable Power Modifier! Now you can:
    1. Choose the Power that will be the Base, and add the Switchable (Base) Power Modifier to it for each alternate set of Powers, it can be selected multiple times.
    2. Choose the Power or Powers that will be Alternate set(s), and assign them the Switchable (Alternate) Power Modifier.
    3. Assign the Base Power for each Switchable Alternate Power. There's a drop-down menu on the Modifier that will list the available Base Powers, simply select the appropriate Base Power.
    4. On the In-Play tab, select the Power(s) that are currently active.
  • Permanent Adjustment selection for SPC Super Power points

Settings

  • Added Starship construction Edge Types to support High Space from StoryWeaver

Background Code Fixes for Settings Authors

  • Added support for a third and fourth Menu item (The third is currently being used by the Super Powers Companion to support the Switchable Power Modifier)
  • The IgnoreRank now incluces a check using new hero-level tags
  • Added the IgnoreRank capability to the Arcane Powers
  • Added a Standard Creature flag to remove Stock Creatures

What changed in version 2.5 of the data files?

Core

  • Linguist Edge languages information disappearing has been fixed
  • A new identity tag group (IgnoreRank) has been added to Edges and Powers to allow abilities that enable heroes to Ignore Rank requirements in specific instances. By simply assigning the IgnoreRank tag for a given Edge or Power to a hero, when that hero chooses Edges or Powers the designated items will show as valid on the drop-down lists regardless of rank requirements.

Fantasy Companion

  • Scamper, Sunder, and Improved Sunder were mistakenly altering the Cash Multiple - this was fixed

What changed in version 2.4 of the data files?

Core

  • Arcane Background - Super Powers: Power-based Skills now count as lower than their linked Attribute for purposes of Advancement
  • Issues with the Linguist Edge and Multiple Language validation messages have been corrected
  • The character sheet now displays the special Notes for those weapons that have them
  • Added Martial Artist to the requirements for the Adept Edge
  • If the Toughness display is too long, it will shorten the label to 'Tough'
  • The Damage/Min Str for Bill Club/Baton has been corrected
  • Under certain circumstances, the weapons and adjustments could end up overlapping each other on the printed output.

Super Powers Companion

  • If the power level is altered (ex. Minor Leaguers, Second String), Super Karma and Natural Growth were adding the wrong amount of power points, and this has been corrected
  • Power Trappings now allow more text by way of using a smaller font
  • Creatures have been set up as Creature Types, and the stock Creatures have been replaced
  • There's now an Attack Power Summary to show the total values for those powers that are used to attack opponents and have variable Damage and/or Range values

Fantasy Companion

  • Magic Item bonuses have been changed to reflect rules clarifications. Artificer Skill Bonuses are now linear bonuses rather than die type increases, and die type modifications are available through the Artifact-level Boost/Lower Trait Power
  • Skills have been added to the list of Traits available for the Boost/Lower Trait Magic Item Power
  • Bodyguard and Sentinel have been added as Creature Types and stock versions have been created

Horror Companion

  • One of the Chosen and Occultist prerequisites have been corrected
  • Some Horror Companion content was not properly flagged as being part of the Horror Companion
